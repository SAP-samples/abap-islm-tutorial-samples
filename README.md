# Intelligent Scenario Lifecycle Management (ISLM) GenAI Tutorial Samples

Find files and artifacts referenced in Intelligent Scenario Lifecycle Management (ISLM) SAP tutorials in one place. Use them directly to follow the tutorials without copying content manually.

Find the mentioned SAP tutorials (filtered for Intelligent Scenario Lifecycle Management) [here](https://developers.sap.com/tutorial-navigator.html?search=islm&tag=tutorial%3Atype%2Ftutorial).
